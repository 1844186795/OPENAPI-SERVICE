# 错误码说明

## 错误响应格式

```json
{
  "code": 40101,
  "message": "Invalid signature",
  "data": null
}
```

## 错误码分类

| 状态码 | 错误码范围 | 说明 |
|--------|-----------|------|
| 400 | 40000 - 40099 | 业务逻辑错误 |
| 401 | 40100 - 40199 | 认证失败 |
| 403 | 40300 - 40399 | 权限不足 |
| 404 | 40400 - 40499 | 资源不存在 |
| 422 | 42200 - 42299 | 请求参数校验失败 |
| 429 | 42900 - 42999 | 请求频率过高 |
| 500 | 50000 - 50099 | 服务器内部错误 |

## 详细错误码

### 认证错误 (401)

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 40100 | 认证失败，无效的 API Key | 检查 `X-App-ID` 是否正确 |
| 40101 | 签名无效（时间戳超时、Nonce 重复、签名不匹配） | 检查签名计算流程 |
| 40102 | API Key 已过期 | 重新申请 API Key |

### 业务错误 (400)

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 40000 | 通用业务错误 | 查看 message 字段获取详情 |

### 资源错误 (404)

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 40400 | 资源不存在 | 检查请求的资源标识符 |

### 参数校验错误 (422)

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 42200 | 请求参数校验失败 | 查看 data 字段获取具体校验错误信息 |

### 服务器错误 (500)

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 50000 | 服务器内部错误 | 请联系管理员 |

## 通用规则

- 所有成功响应 `code` 均为 `0`
- `message` 字段提供人类可读的错误描述
- `data` 字段在错误时可能包含额外的错误详情
- 每个请求的响应头包含 `X-Request-ID`，排查问题时请提供此 ID
