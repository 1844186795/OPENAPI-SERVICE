# 系统架构总览

## 项目简介

OpenAPI Service 是一个基于 FastAPI 构建的对外 API 服务平台，专注于提供安全、稳定、高效的接口服务。

## 技术栈

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 编程语言 | Python 3.12+ | 使用 async/await 异步编程 |
| Web 框架 | FastAPI | 高性能异步框架，自动生成 OpenAPI 文档 |
| 数据库 | MySQL 8.0+ | 关系型数据库 |
| ORM | SQLAlchemy 2.0 | 异步引擎 (async engine) |
| 数据校验 | Pydantic v2 | 请求/响应模型校验 |
| 数据库迁移 | Alembic | 异步迁移支持 |
| 项目工具 | uv | Python 虚拟环境和依赖管理 |

## 请求生命周期

1. 客户端发起 HTTP 请求
2. 请求日志中间件记录请求信息，生成 Request ID
3. 认证中间件（如路由需要认证）验证 API Key 签名
4. 路由处理函数接收并校验请求参数
5. Service 层执行业务逻辑
6. 数据访问层通过异步 ORM 操作数据库
7. 统一响应格式返回结果
8. 异常处理器捕获并按统一格式返回错误
